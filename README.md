**Inventario Tree-Stock** es una aplicación de consola desarrollada en **Java** que implementa un sistema de inventario utilizando una estructura de datos de **Árbol Binario de Búsqueda (BST)**. El sistema permite organizar productos de forma jerárquica y eficiente mediante el uso de recursividad y gestión de referencias.

---

## 🎯 Objetivo del Proyecto
Desarrollar una solución técnica para el registro y búsqueda de productos, optimizando el rendimiento mediante estructuras no lineales y garantizando que el inventario se mantenga ordenado automáticamente por su ID.

## 🛠️ Arquitectura del Software
El proyecto sigue una estructura modular dividida en tres clases principales:

| Clase | Función | Descripción |
| :--- | :--- | :--- |
| `Producto.java` | **El Nodo** | Contiene los atributos del producto (`id`, `nombre`) y los punteros `izquierdo` y `derecho`. |
| `ArbolInventario.java` | **La Lógica** | Implementa algoritmos recursivos para la inserción, búsqueda y recorrido *Inorden*. |
| `Main.java` | **La Interfaz** | Gestiona el menú interactivo y la comunicación con el usuario. |
